Causal inference example

Jan 1, 0001 7 min read

Introduction

In this example we explore “generalized random forests” (grf) to estimate treatment effects in the setting of a randomized experiment.

Generalized Random Forests (grf) is a package based on a paper by Susan Athey, Julie Tibshirani, Stefan Wager. grf provides methods to estimate treatment effects.

The algorithm is comparable to “normal” random forests, in the sense that it estimates an outcome based on the average estimation in many regression trees. There are a few differences with a normal random forest:

  • In a normal random forest the splitting in the regression trees is based on minimizing the “Root Mean Square Error” (RSME), while in grf splitting is based on maximum heterogeneity of the treatment effect.
  • In addition to a normal random forest, grf uses a subsample of the data in to construct the trees, while the other part of the data is used to estimate the treatment effect based on the constructed trees.

Source

The attractive part of grf is that it:

  • provides estimates of the individual treatment effect
  • with confidence intervals
  • estimates average treatment effects (ate)
  • estimates conditional treatment effects (cate)

In this notebook we will run a grf on simulated data and test how well grf performs.

Please note that we have assumed in the data that treatment assignment was random.

Data

First we construct a list with 10000 persons and make a data frame of it.

ID <- 1:10000
df <- as.data.frame(ID)

We will add the following variables:

  • age
  • having a chronic disease (0,1)
  • SES (1,2,3)
  • Treatment (0,1)
  • Noise
  • Treatment_effect (as a function and age)
  • HC as a function of age, chronic, SES and treatment
set.seed(123)
df <- df %>%
  mutate(age= round(abs(rnorm(10000,45, 50)),0)) %>%
  mutate(Chronic = rbinom(10000, 1, 0.05)) %>%
  mutate(SES = sample(c(1,2,3), size = 10000, replace = T)) %>%
  mutate(Treatment = sample(c(0,1), size= 10000, replace = T)) %>%
  mutate(Noise = rnorm(10000, 0, 1000)) %>%
  mutate(Treatment_effect = -600  - age^2 - age * SES*100 - Chronic*1000 ) %>%
  mutate(HC = 10000 + age^2 + 3000* Chronic + 4 * SES + Treatment_effect * Treatment + Noise )

Analysis with generalized random forests

Plan of action

  1. We will split the data in a train and test set
  2. We will run a grf analysis to estimate the individual treatment effect
  3. We will compare the estimated individual treatment effect versus the real treatment effect
  4. We will try to identify the most important variables that drive the estimation of the individual treatment effect by:
    1. the variable importance by the grf package
    2. a linear regression of the predictors on the estimated treatment effect
    3. a normal random forest of the estimated treatment effect on the predictors
  5. We will compare the estimated average treatment effect with the real average treatment effect

Split the data

Train Test

set.seed(123)
trainIndex <- createDataPartition(df$HC, p = .5,
                                  list = FALSE,
                                  times = 1)
dfTrain <- df[ trainIndex,]
dfTest  <- df[-trainIndex,]

We will then construct a dataframe with the co_variates \(X\).

X_train <- dfTrain %>%
  select(age, Chronic, SES)

X_test1 <- dfTest %>%
  select(age, Chronic, SES)

Change train en test data frames in matrices

\(X\) co-variates \(Y\) Outcome \(W\) Treatment

X <- as.matrix(X_train)
Y <- as.matrix(dfTrain$HC)
W <- as.matrix(dfTrain$Treatment)

X_test <- as.matrix(X_test1)

Estimate heterogenous treatment effects

Run a causal forest (without tuning)

set.seed(123)
cf <- causal_forest(X,Y,W, num.trees = 5000)
pred <- predict(cf, X_test, estimate.variance = TRUE)

Add predictions and confidence intervals to the test set

dfTest <- dfTest %>%
  mutate(Predictions_cf = as.vector(pred$predictions)) %>%
  mutate(Variance = as.vector(pred$variance.estimates))%>%
  mutate(Lci = Predictions_cf - 1.96* sqrt(Variance)) %>%
  mutate(Uci = Predictions_cf + 1.96* sqrt(Variance))

Compare estimated individual treatment effects to the real individual treatment effects

ggplot(data=dfTest, aes(x=Treatment_effect, y=Predictions_cf)) + geom_point(color="blue")  + theme_classic() +geom_linerange(aes(ymin=Lci, ymax=Uci))+ geom_abline(slope = 1 , intercept = 0, col="green") + xlim(-120000,0) + ylim(-120000,0)

Causal forest seems to predict fairly well

The correlation between the prediction and the actual treatment effect is 99%.

Random Forest

set.seed(123)

rf_trainIndex <- createDataPartition(dfTest$ID, p = .8,
                                  list = FALSE,
                                  times = 1)
rfTrain <- dfTest[ rf_trainIndex,]
rfTest  <- dfTest[-rf_trainIndex,]
set.seed(123)
rfcontrol <- trainControl( method = "cv",
                           number = 10
                           )


rf <- train(Predictions_cf ~ age + Chronic + SES, 
                data = rfTrain, 
                method = "ranger",
                importance = "permutation",
                trControl = rfcontrol)
## note: only 2 unique complexity parameters in default grid. Truncating the grid to 2 .
plot(varImp(rf))
